# Mesto. Валидация форм, работа с API, сборка Vite, GitHub Pages

Интерактивная страница для обмена фотографиями. Реализована возможность добавлять и удалять карточки, ставить лайки, редактировать профиль и аватар, просматривать статистику пользователей.

## Технологии
- HTML5, CSS3, JavaScript (ES6)
- Валидация форм на стороне клиента
- Работа с REST API (сервер `mesto.nomoreparties.co`)
- Асинхронные запросы (Promise, `fetch`)
- Модульная структура (ES Modules)
- Сборка — Vite
- Деплой — GitHub Actions (автоматическая публикация в публичный репозиторий)

## Функциональность
- Загрузка карточек и данных пользователя с сервера
- Редактирование профиля и аватара
- Добавление / удаление карточек (удаление доступно только автору)
- Постановка и снятие лайка, отображение количества лайков
- Валидация всех форм с кастомными сообщениями
- Модальное окно подтверждения удаления карточки
- Статистика пользователей (по клику на логотип):
  - общее количество карточек
  - дата первой и последней карточки
  - количество уникальных пользователей
  - максимальное количество карточек у одного пользователя
  - список всех пользователей с количеством карточек
